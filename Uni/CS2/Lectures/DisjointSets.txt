Disjoint Sets:
	â€¢ Two or more sets with no common elements are called disjoint sets. Eg, S1 = {1,2,3}, S2 = {4,5}
	â€¢ As S1 âˆ© S2 = Ã˜ Empty set (S1 and S2 have nothing in common), we can call them a disjoint set
Disjoint Sets Usage:
	â€¢ It can help keep track of the set an element belongs to
	â€¢ It is easier to check whether two given elements belong to the same subset or not (Find Operation)
	â€¢ Eg, S1 = {1,2,3,4} S2 = {5,6,7}
	â€¢ find(1) == find(2) will return true as both 1 and 2 belong to the same set
	â€¢ They can help merge 2 sets into 1 (Union Operation)
	â€¢ Eg, S1 = {1,2,3,4} S2 = {5,6,7}
	â€¢ Union will be {1,2,3,4,5,6,7}
	â€¢ In a disjoing set, Union(2,6) will give you the same set as something like Union(1,5)
	â€¢ Union(2,6) will find the set where 2 belongs to and finds the set where 6 belongs to and performs union between the sets, similarly Union(1,5) will do the same with 1 and 5
Representative:
	â€¢ Each set is represented by an element in the set, EG:
		- {1},{2},{3},{4},{5}, in the list of sets, each number is the representative of its own set
Disjoint Set Operations:
	â€¢ Given a disjoint set, we can edit them using the union operation
	â€¢ EG:
		- union(1,3) would make the previous structure look like {1,3},{2},{4},{5}
		- Here we designate either 1 or 3 as the marker, choosing 1
		- If the following operations are carried out: union(1,4) and union(2,5) (assuming 2 is marked)
		- The structure will be updated to {1,3,4}, {2,5}
		- Now findset operation can also be performed, findset(3) should return 1, as 1 is the marked element in the set that contains 3
